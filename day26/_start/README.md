### Initial State
When you start Day 26, you're likely:
- **Familiar** with collections and loops
- **Ready** to learn functional programming
- **Curious** about streams and lambdas
- **Unsure** about when to use streams vs loops

### Common Starting Challenges

1. **Lambda Syntax**
   - Arrow notation (->)
   - Parameter types
   - Return values
   - **Tip**: (parameter) -> expression or { statements }

2. **Stream API**
   - Creating streams
   - Intermediate operations
   - Terminal operations
   - **Tip**: Stream operations are lazy until terminal operation

3. **Functional Thinking**
   - Declarative vs imperative
   - Method chaining
   - Immutability
   - **Tip**: Think "what" not "how"
