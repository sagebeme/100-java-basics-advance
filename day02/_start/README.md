### Initial State

### Common Starting Challenges

1. **String Method Overload**
   - So many methods: `toUpperCase()`, `toLowerCase()`, `substring()`, `indexOf()`, etc.
   - Which one to use when?
   - **Tip**: Start with the most common ones, others will come naturally

2. **Type Conversion Confusion**
   - When does conversion happen automatically?
   - When do you need explicit casting?
   - What's the difference between `parseInt()` and `valueOf()`?
   - **Tip**: Automatic = widening (int‚Üídouble), Manual = narrowing (double‚Üíint)

3. **String Formatting Complexity**
   - `printf()` vs `println()` vs `format()`
   - Format specifiers (`%d`, `%f`, `%s`)
   - Decimal precision formatting
   - **Tip**: Start with `printf()` for formatted output, it's very useful

4. **Mathematical Operations**
   - Integer division vs floating-point division
   - Order of operations
   - Type promotion in calculations
   - **Tip**: Be explicit with types in calculations to avoid surprises
