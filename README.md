## ğŸ“ How to Use This Repository

1. **Clone the repository**
   ```bash
   git clone git@github.com:sagebeme/100-java-basics-advance.git
   cd 100-java-basics-advance
   ```

2. **Set up your development environment**
   - Install JDK 11 or higher
